Get started!

  • How to start & play the game
    Run "touchthegirl.exe" and follow the instruction for making save data. (1st time only) The storage settings panel will appear, and follow the instructions below.



    1. Move the slider to the far right.
    2. Press "Close" to close the panel.
    3. Press "OK". The title screen will be displayed.

    Press "START" to start the game. (If you've any problem with starting the game, please see FAQ&trouble shooting.)
    Click(tap) or drug the parts you want to touch. Touch guide shows you operation methods with "Touch icon" when you place the cursor over(or tap on the touch screen) touchable points.
    (You can play this game on touch screen devices as well, although multi-touch is not supported.)

    ** Multiple launches - This game is available multiple launches (windows), but please note that the settings data of the last closed window is stored. If you don't want to save the settings data, forcibly terminate the game from the task manager.
  • About installing additional costume sets
    You can install and play with additional costume sets DLC for this game. For more information on downloading additional costumes, please visit Sawatex website.
    ** Please note that the costume set files are different between "Soft version" and "Hard version" of the game.

    -- For the paid costume sets : Copy all the files in the package to the "costumes" folder in the game (touchthegirl) folder (or overwrite them when updating), and then run the attached installer. The files will not be copied automatically.
    -- For the free costume set : You don't need to copy the files, just run the installer. Once installed, there is no need to update individually.

    Please read the "README1st.txt" in the costume set package for more information.
  • About the update of the game and additional costume sets
    This game will be updated on an irregular basis. When updating the game and additional costume sets, please overwrite all the files and folders in the original folder. (There is no installer or updater.)
    ** Please be sure to OVERWRITE all the files and folders. The new package of the game doesn't contain the paid additional costume files, so If you've already installed these costume sets, launching the game from there will cause a loading error. And if you accidentally launch the older version of the game, it may cause problems with your saved data. so we recommend that you don't leave older versions behind. (** For the free costume sets, you don't need to update.)
    Please see the page below for more information and history about updates.
    Sawatex website - Upgrade information
  • About MOD
    You can incorporate the mods you create yourself for some parts in Hard version. Please refer the page below for more information. (Not a paid option)
    https://www.patreon.com/posts/mod-47618643
  • Uninstall
    Quit the game, search "#sawatex" folder on your system drive and delete it. (Save data is deleted.Please notice if you installed other our works, also their save files are deleted.)
    And delete the folder installed this game. It doesn't use such as the registry or any other system files.
  • Operation on OS other than Windows
    You can play this game by running "touchthegirl.swf" with official swf stand alone player provided by Adobe. Other 3rd party tools or add-ons may be available, but we can not guarantee they will work correctly.

The Device

Pressing the "Device" button(or X) expands the player's "Device" at the bottom of the screen. Using the Device allows you to perform different actions depending on the zoom level, state of the clothing, etc.
For instructions on how to use the Device, please refer to Touch guide that appears when you hover the mouse over the Device.

Device controls
When the device action is triggered, the "Device controls" is displayed in the lower right corner of the screen, and you can select a further action by pressing each button.
Button types and functions differ depending on each action. Please refer to the display in the Touch guide. You can also select the window type from the Global settings.

Ecstasy mode
By pressing the Ecstasy mode button (heart-shaped), you can change the girl's reaction while activating the Device action. And you can also change it automatically over time from the Global settings.
Discharge Now button
After selecting the various "Discharge" buttons on the Device controls, the "Discharge Now" button will appear following. (Some discharge actions may not be displayed but are executed immediately.) Click on it to execute the discharge action immediately.

If it's not pressed, The discharge action will be automatically executed at the normal timing, but if you turn on the "repeat button" at the top, it'll repeat the discharge preliminary motion until the "Discharge Now" button is pressed.
You can also return to the previous state before selecting the Discharge buttons with the "back button" at the bottom.

FAQ&trouble shooting

  • If you have troubles that "impossible to start", "error message is displayed" or "no sounds", These are mostly caused by false positives of your security software, which may uniformly ban swf (Flash) files even though they are harmless. (ex. Avast, AVG) If this is the case, please try running the file with your anti-virus temporarily turned off.

  • For installing the paid additional costume sets, you'll need to manually copy the files into the "costumes" folder. If you get an error message about the costume set, please make sure that the files are in the "costumes" folder.

  • If any settings are not saved, or additional costumes are not displayed after installing them, the "Shared Object File" (commonly known as Flash-Cookie: a file used by Flash programs to save settings) is not working properly.
    Also in this case, your security software or browser add-on may prohibit to access Flash cookie files. Please confirm their settings.

Please refer the page below for more details or other cases.
https://sawatex.x0.com/faq.htm
